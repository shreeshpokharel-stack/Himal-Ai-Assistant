# 🤖 HimalAI Assistant

A desktop automation assistant built with Python that can launch applications, open websites, and execute system commands through simple text-based instructions.

## 🚀 Features

- Open Google Chrome
- Open Spotify
- Open WhatsApp
- Open Notepad
- Open Calculator
- Open Paint
- Open Command Prompt (CMD)
- Open websites
- Google search fallback for unknown commands
- Shutdown and restart commands

## 🛠️ Technologies Used

- Python
- psutil
- subprocess
- os
- webbrowser

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd Himal-AI-Assistant
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the assistant:

```bash
python main.py
```

## 💻 Example Commands

```text
open chrome
open spotify
open whatsapp
open notepad
open calculator
open paint
open cmd
youtube
google
shutdown
```
